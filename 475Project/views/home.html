Instructions

Viewing/Filtering Students

  1. Select "Students" in top navbar
  2. Click on a student's name to reveal more information, such as the student's schedule, grades, and tags
  3. Use the search box to filter by tags

Adding Students

  1. Select "Add Student" from the top navbar
  2. Fill in the student's information using the form
  3. Click the "Submit" button to add the student to the system

Editing Students

  1. Select "Student" from the top navbar
  2. Click on "Edit" to the right of the student's name you want to edit
  3. Change information in the student's form
  4. Click the "Submit" button to save your changes

Viewing/Editing Courses and Sections

  1. Select "Courses" in top navbar
  2. Click "Edit" to the right of the course whose information/section data you want to edit
  3. Change information in the course's form
  4. Click the "Submit" button to save your changes

Adding Courses

  1. Select "Add Course" from the top navbar
  2. Fill in the course's information using the form
  3. To add a course section, click "Add Section"
  4. Click the "Submit" button to add the course to the system

Assigning TA/LA Candidates for Courses

  1. Select “Courses” from the top navbar
  2. Find your desired course (each is listed in their own table)
  3. Click on the button labeled “Assign Candidates” listed to the right of your desired section
  4. Use the filters (search boxes and buttons) to search through the list of students
  5. Once a student is found, click the green button labeled “Assign Candidate” listed in that student’s row

Viewing TA/LA Candidates for Courses

  1. Select “Candidates” from the top navbar

Assigning Students as TAs/LAs for Courses

  1. Follow instructions for “Assigning TA/LA Candidates for Courses” for the students
  2. Select “Candidates” from the top navbar
  3. The candidates you selected in step one are now visible on this page
  4. Click the green button labeled “Finalize Assignment” in the rows of the desired students

Viewing TA/LA Assignments for Courses

  1. Select "Final Assignments" from the top navbar

Viewing Student Workload

  1. Select “Final Assignments” from the top navbar
  2. Select the “Students” tab below the page header

Exporting TA/LA Assignments and Student Workload as CSV Files

  1. Select “Final Assignments” from the top navbar
  2. Select the either the "Courses" or “Students” tab below the page header to select the table you would like to export
  3. Click the blue export button on the top right of the page to download the table as a CSV file
